## Hypothesis Test 
**Null Hypothesis (Ho):**
μ = 25  
**Alternative Hypothesis (Ha):**
μ < 25  

## Test Statistics
$\[ T= \frac{\bar{x} - \mu_0}{S/ \sqrt{n}} = \frac{20.3 - 25}{5/ \sqrt{21}} = -4.308 \]$
T = (x̄ - μ₀) / (s / √n)  
   = (20.3 - 25) / (5 / √21)  
   = -4.308
