## a
1. $H_0$ : $\mu $= 25, $H_a$: $\mu$ < 25
   
2. set $\alpha$= 0.05
   
3.
$$
t_{\text{STAT}} = \frac{\bar{X} - \mu_0}{S / \sqrt{N}} \sim t_{(N - 1)} \quad → \quad t_{\text{STAT}} = \frac{\bar{X} - 25}{S / \sqrt{N}} \sim t_{(21 - 1)} \quad → \quad t_{\text{STAT}} = \frac{\bar{X} - 25}{S / \sqrt{N}} \sim t_{20}
$$

4. $t$ ≒　-4.31

5. $t_{20,0.05} = -1.725 $
  
6. Because $t = -4.31$ $<$ -1.725  → reject $H_0$
