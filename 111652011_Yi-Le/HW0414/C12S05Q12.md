## Solution of a

The regression equation is given by: $\hat{y} = a + bx$  

$\bar{x} = 3$  
$\bar{y} = 8.8$  
$S_{xx} = 10$  
$S_{xy} = 15$  
$b = \frac{S_{xy}}{S_{xx}} = 1.5$  
$a = \bar{y} - b \bar{x} = 4.3$  
Thus, $\hat{y} = 4.3 + 1.5x$
